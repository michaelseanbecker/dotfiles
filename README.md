# Dotfiles

https://www.chezmoi.io/quick-start/#start-using-chezmoi-on-your-current-machine

## Usage

sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply $GITHUB_USERNAME

## Editing
To open the dotfile repo in VSCode, run 
```
chezmoi edit 
 ```
